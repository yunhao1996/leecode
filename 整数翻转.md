问题：给出一个 32 位的有符号整数，你需要将这个整数中每位上的数字进行反转。
```python

示例 1:
输入: 123
输出: 321

示例 2:
输入: -123
输出: -321
 
示例 3:
输入: 120
输出: 21
```
答案：
```python
class Solution(object):
    def reverse(self, x):
        """
        :type x: int
        :rtype: int
        """
        s = str(x)[::-1].rstrip('-')
        if int(s) < 2**31:
            if x >= 0:
                return int(s)
            else:
                return 0-int(s)
        return 0
        
```
